# Project Brief: PromptRefinery

PromptRefinery is a web-based AI prompt engineering tool designed to help users transform vague instructions into structured, high-quality prompts for LLMs (like ChatGPT, Claude, Llama).

## Core Objectives
- **Enhance Prompt Quality:** Use advanced LLM technology (UncloseAI Hermes) to rewrite prompts with better context, specificity, and structure.
- **Educate Users:** Provide real-time quality scoring and actionable feedback to help users understand prompt engineering principles.
- **Manage Resources:** Offer a curated library of templates and a personal system for saving and managing enhanced prompts.

## Key Features
- **Intelligent Enhancement:** Automated rewriting of prompts to improve clarity and effectiveness.
- **Creativity Controls:** Adjustable temperature settings (Precise, Balanced, Creative, Max).
- **Real-time Scoring:** Instant analysis of prompt quality metrics.
- **User Management:** Secure authentication (PHP/SQLite) for saving and retrieving personal prompt history.
- **Prompt Library:** A collection of pre-made templates for various use cases (Business, Coding, Creative Writing).

## Technology Stack
- **Frontend:** Vanilla HTML5, CSS3, JavaScript.
- **Backend:** PHP (7.4+).
- **Database:** SQLite3.
- **AI Integration:** UncloseAI Hermes Model.

## Significance
PromptRefinery bridges the gap between casual AI use and professional prompt engineering, empowering developers, writers, and business professionals to extract maximum value from AI models through precision and structure.
